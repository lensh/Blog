# 浏览器从输入URL 到页面加载完成的过程

  1. ⽤户输⼊ baidu.com
  2. 浏览器通过DNS，把域名解析为IP，参考 dns递归查询和迭代查询
  3. 和IP地址建⽴TCP链接 发送HTTP请求
  4. 服务器接收请求，查库，读⽂件等，拼接好返回的HTTP响应
  5. 浏览器收到⾸屏html，开始渲染
  6. 解析html为DOM tree
  7. 解析css 为CSSOM tree
  8. dom + css ⽣成render-tree，绘图
  9. 加载script的js⽂件
  10. 执⾏js
